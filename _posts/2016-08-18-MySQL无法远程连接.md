title: MySQL无法远程连接
---

* 1.**修改my.cnf, 默认在/etc/mysql/my.cnf**  

    找到bind-address修改如下:

```shell
bind-address = 0.0.0.0
```

* 2.**修改用户权限**

    MySQL默认建用户的时候，回指定host为127.0.0.1/localhost,修改为%

```shell
$ mysql -uroot -p密码
mysql> use mysql;
mysql> update user set host=' %' where user=' root';
```
    （注意上面引号中的空格不能丢）

* 3.**重启MySQL**

```shell
$ service mysql restart
```
