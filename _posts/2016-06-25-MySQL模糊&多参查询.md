* 使用LIKE查询
<br>例子：
```
<select id="search" parameterType="list" resultType="com.mr6.model.File">
    SELECT
        *
    FROM
        files
    WHERE
        `name` LIKE
    <foreach collection="list" item="item" index="index" open="" separator="OR `name` LIKE" close="">
        "%“#{item}"%"
    </foreach>;
</select>
```
